Ames_housing_data.csv contains the raw ames housing data which has been cleaned in the provided python notebook. 
Ames Final Data contains the csv with the final cleaned data with no outliers, missing values and dummy variables for categorical data evaluation.
Ames housing feature description contains a brief description about the various features mentioned in the Ames_housing_data.csv.
final_model.joblib and scaler.joblib are joblib files, containing the final_mode with the best hyperparameters obtained using GridSearchCV for the elastic net and scaler which scales data to prevent differences in units from unfairly shaping the regression line.
